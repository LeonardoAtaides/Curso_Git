GIT E GITHUB

Branch: são as ramificações de código criadas(versão especifica) sendo possível agrega-la na versão principal

Comandos Básicos para o terminal

git init -> serve para inicializar o git
criando os arquivos necessários para inicializar repogit 

git status -> mostra so branchs e os commits 

git add -> adiciona o arquivo, para salvamento prévio
e com "." adiciona todos os aquivos -> git add .

git commit: Salva o arquivo
git commit -m '' -> add um comentário

clear(limpar) -> Serve para limpar o terminal

Cd(change Directory) -> muda o local do diretório atual, para isso utilize: 
cd<caminho> entre ""

posso copiar e colar o caminho da pasta
ou digitar

ls(list) -> Lista o que está dentro do diretório
Ex:
ls- al(lista arquivos ocultos)

SSH - é um protocolo de rede criptográfica para operar serviços de rede com segurança em uma rede não segura.

*COM O GIT VOCÊ PODE CLONAR UM PROJETO(BAIXAR DE UMA EMPRESA) DO GITHUB OU SUBIR UM PROJETO, VAI VARIAR DE ACORDO COM OQUE
VOCÊ PRECISA FAZER*

CLONANDO:
ABRE O PROJETO E CLICA EM <CODE>
E DEPOIS SSH E COPIA O LINK

Ex:
git clone<projeto_git>
Com o link:

pedira a senha do ssh criado por você.
-> Assim ele copia todos os arquivos do repositório para a pasta selecionada
incluindo a pasta

-> para trazer apenas os arquivos usa-se:
git@github.com:LeonardoAtaides/MyProjects.git .
                                             |-> apenas add com o espaço o ponto

SUBINDO:

git push -> empurrar irá eviar as alterações ao GitHub
